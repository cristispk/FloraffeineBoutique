# Wireframe — Public Home Shell

## Desktop

- Sticky header: logo left, primary nav center/right, auth CTA group right.
- Hero block immediately under header with title, subtitle, primary CTA, secondary CTA.
- Three premium content sections with generous spacing.
- Trust/testimonial strip.
- Structured footer with 3-4 columns and legal line.

## Mobile

- Header with logo + drawer toggle.
- Drawer contains all public nav links and auth CTAs.
- Hero and sections stack vertically with consistent spacing.
- Footer collapses into stacked columns.
