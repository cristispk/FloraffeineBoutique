# Wireframe — Public Generic Content Shell

## Desktop

- Sticky public header.
- Optional page intro band (`public_hero`) with title and short subtitle.
- Main content in readable container with section blocks.
- Optional CTA banner before footer.
- Structured footer.

## Mobile

- Header with drawer navigation.
- Intro band compacted with same hierarchy.
- Content sections stacked with medium spacing.
- Footer stacked.
