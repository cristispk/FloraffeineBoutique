# Wireframe — Admin Detail/Review Shell

## Desktop

- Admin sidebar and topbar persist.
- Header with breadcrumb trail and page actions.
- Main two-column layout:
  - left: detail fields and status blocks
  - right: action panel (approve/reject/suspend/reactivate)

## Mobile

- Drawer navigation.
- Single-column detail flow.
- Action panel moves below details with full-width buttons.
