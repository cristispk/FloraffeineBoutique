# Wireframe — Mobile Navigation States

## Public shell

- Closed: logo + drawer toggle.
- Open: full-height drawer with public links and auth CTAs.
- Active link highlighted via route-state style.

## App shell

- Closed: topbar with drawer toggle, page title.
- Open: app sidebar links in drawer, grouped by role.
- Profile/logout visible in topbar or drawer footer.

## Admin shell

- Closed: compact topbar with drawer toggle and context title.
- Open: admin operational links in drawer.
- Current section highlighted; logout reachable without scrolling excessively.
