# UI Visual Standards

## Source of Truth
When a design source-of-truth exists, it is mandatory.

## Mandatory Requirements
- exact palette extraction
- explicit visual token mapping
- visual fidelity over approximation
- reusable components
- no page-specific hacks
- no inline styles in Blade pages

## Visual Review Criteria
- logo sizing
- card proportions
- spacing rhythm
- hierarchy
- consistency
- source-of-truth fidelity
